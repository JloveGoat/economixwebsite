# Images Folder

Place your website images here. The following placeholders are referenced in the site:

## Home Page
- `hero-bg.jpg` — Hero section background image (recommended: 1920×1080, landscape)
- `hero-illustration.png` — Main hero illustration or product screenshot (recommended: 800×600)
- `stat-icon-1.svg` / `stat-icon-2.svg` / `stat-icon-3.svg` — Stat section icons
- `feature-1.jpg` — First feature/highlight image (recommended: 600×400)
- `feature-2.jpg` — Second feature/highlight image (recommended: 600×400)

## About Us Page
- `about-hero.jpg` — About page hero image (recommended: 1200×600)
- `team-1.jpg` — Team member photo (recommended: 400×400, square)
- `team-2.jpg` — Team member photo
- `team-3.jpg` — Team member photo
- `team-4.jpg` — Team member photo
- `office.jpg` — Office / workspace photo (recommended: 900×600)

## Camps Page
- `camp-1.jpg` — Camp/program cover image (recommended: 600×400)
- `camp-2.jpg` — Camp/program cover image
- `camp-3.jpg` — Camp/program cover image
- `camps-hero.jpg` — Camps page hero image (recommended: 1200×500)

## General / Misc
- `logo.png` — EcoNomix logo (if you prefer an image logo over the SVG text logo)
- `partner-1.png` through `partner-5.png` — Partner/sponsor logos (recommended: 200×80, transparent PNG)

Supported formats: `.jpg`, `.jpeg`, `.png`, `.webp`, `.svg`, `.gif`
